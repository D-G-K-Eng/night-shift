# NIGHT SHIFT

Top-down city driver. Pick up fares, earn cash, dodge the cops.

## Status

**In development.** Built by Hammerz + Viv.

- Game layer: Viv (city grid, car physics, fare system, cop AI)
- Audio engine: Hammerz (synthwave procedural — engine osc, city ambience, pickup/dropoff jingles, siren, wanted level layers)

## Play

Coming soon: https://d-g-k-eng.github.io/night-shift/

## Tech

Single HTML file. Web Audio API + Canvas 2D. No external dependencies.
